ЛИТУИЗМ & ПСИХОДЕЛИЧЕСКИЙ АСЕМБЛЕР ОТ БЕЛКИ
ЖМИ НА КНОПКУ И ПОЛУЧИ НАХУЙ ЛИТУИЗМ!

Мини-пример NASM с пояснением

section .data
  hello db 'Литуизм - это пиздец!', 0Ah

section .text
  global _start

_start:
  mov edx, 23       ; длина строки
  mov ecx, hello    ; адрес строки
  mov ebx, 1        ; stdout
  mov eax, 4        ; sys_write
  int 0x80          ; вызов ядра

  mov eax, 1        ; sys_exit
  int 0x80

Этот код выводит строку «Литуизм - это пиздец!» в терминал Linux x86 через системный вызов sys_write. Конкретно:

  • mov edx, 23 — длина строки
  • mov ecx, hello — указатель на строку
  • mov ebx, 1 — файловый дескриптор stdout
  • mov eax, 4 — номер системного вызова write
  • int 0x80 — вызов ядра Linux

Это классика x86 ассемблера, простой и понятный пример для начинающих литуистов и безумцев.
